This is a pretty interesting announcement.

Google released an agent-friendly CLI to their services, available [here on Github](https://github.com/googleworkspace/cli).

This is first-class support for Google Workspace configuration and access, and ships with over [100 agent skills](https://github.com/googleworkspace/cli?tab=readme-ov-file#ai-agent-skills).

I have to imagine this unlocks a lot of interesting possibilities, as autonomous agents like OpenClaw can start to interact with these services more reliably and make more interesting productivity workflows possible.
